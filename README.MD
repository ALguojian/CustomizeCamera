# 自定义相机

针对小店业务进行扩展，添加拍照时的框框,以及预览页面


1.Add it in your root build.gradle at the end of repositories:

```
allprojects {
        repositories {
            ...
            maven { url 'https://jitpack.io' }
        }
    }
```

2.Add the dependency

```
dependencies {
            implementation 'com.github.ALguojian:CustomizeCamera:2.1'
}
```
